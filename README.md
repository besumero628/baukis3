# Baukis3 - 顧客管理システム

## 説明
Ruby on Rails6 実践ガイド (出版：インプレス　著者：黒田 努)
上記を題材に学習を進め、構築した顧客管理システムです。

## 推奨されるシステム環境
* Ruby 2.6.4
* Rails 6.0.4
* PostgreSQL 11.5
